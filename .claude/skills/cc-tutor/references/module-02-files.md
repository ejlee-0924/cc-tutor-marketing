# Module 2: 파일 다루기 + 스마트 조작법

> 공식 문서: https://code.claude.com/docs/ko/tools

## WHY

{name} {title}님, 지난달 삼성전자 제안서 파일이 어디 있었는지 바로 기억나시나요?

B2B 마케터의 업무 폴더에는 수십 개의 제안서, 콘텐츠 레퍼런스, 고객사 자료가 쌓여 있습니다. 필요한 파일을 찾고, 새 문서를 만들고, 기존 문서를 수정하는 것 — 이것이 Claude Code의 가장 기본적인 능력입니다.

**Before:** 폴더를 하나하나 뒤져서 파일 찾기, 빈 문서에서 처음부터 작성
**After:** "삼성전자 제안서 찾아줘", "LG 마케팅 캠페인 기획서 템플릿으로 새 문서 만들어줘"

## EXPLAIN

Claude Code는 컴퓨터의 파일을 직접 다룰 수 있습니다. 핵심 도구 3가지:

### 1. 파일 읽기 (Read)

> 📖 비유: "프로젝트 폴더에서 보고서 꺼내 읽기"

Claude에게 파일을 보여달라고 하면 내용을 읽어서 보여줍니다.

    이 파일 읽어줘: proposals/samsung-dt-2026.md

### 2. 파일 쓰기 (Write / Edit)

> 📖 비유: "새 문서 작성해서 저장하기"

새 파일을 만들거나 기존 파일을 수정할 수 있습니다.

    campaigns 폴더에 lg-ai-literacy.md 파일을 만들어줘

### 3. 파일 검색 (Glob / Grep)

> 📖 비유: "폴더에서 특정 파일이나 내용 검색하기"

파일 이름으로 찾거나, 파일 안의 내용으로 검색할 수 있습니다.

    campaigns 폴더에서 "AI 교육" 관련 파일 찾아줘
    모든 캠페인 기획서에서 "AI 리터러시" 키워드가 들어간 부분 찾아줘

### 핵심 정리

| 도구 | B2B 비유 | 예시 |
|------|---------|------|
| Read | 보고서 꺼내 읽기 | "이 제안서 내용 보여줘" |
| Write/Edit | 새 문서 작성/수정 | "캠페인 기획서 초안 만들어줘" |
| Glob | 파일명으로 검색 | "campaigns 폴더에 뭐가 있어?" |
| Grep | 내용으로 검색 | "'삼성전자' 들어간 파일 찾아줘" |

---

파일 다루기 외에도, Claude Code를 **더 똑똑하게 조작하는 방법**들이 있습니다:

### 4. 스크린샷으로 설명하기

> 📖 비유: "이거 좀 봐줘" 하면서 화면을 보여주는 것

Claude Code 대화창에 **스크린샷을 붙여넣기(Ctrl+V / Cmd+V)** 할 수 있습니다. 에러 화면, 참고할 자료, 경쟁사 캠페인 등을 캡처해서 바로 보여주면 말로 설명하는 것보다 훨씬 정확합니다.

    [스크린샷 붙여넣기 후] 이 캠페인 레퍼런스 참고해서 비슷한 구조로 기획서 만들어줘

### 5. Plan 모드 — 실행 전에 계획부터

> 📖 비유: "바로 하지 말고, 먼저 어떻게 할 건지 계획 보여줘"

큰 작업을 시킬 때, Claude가 바로 실행하면 의도와 다를 수 있습니다. **Shift+Tab을 2번** 누르면 **Plan 모드**로 전환됩니다. 이 모드에서는 Claude가 먼저 계획을 보여주고, 승인 후에만 실행합니다.

```
일반 모드:  "해줘" → 바로 실행
Plan 모드:  "해줘" → 계획 제시 → 승인 → 실행
```

### 6. "Think harder" — 더 깊이 생각하게 하기

> 📖 비유: "대충 하지 말고 진지하게 고민해봐"

복잡한 문제에서 Claude의 답이 피상적일 때, **"think harder"** 또는 **"깊이 생각해봐"**라는 키워드를 넣으면 Claude가 더 심층적으로 분석합니다.

    이 캠페인 플로우가 타겟 기업의 Pain Point를 제대로 해결하는지 think harder 해서 분석해줘

### 7. 대화 흐름 제어 — 중단·요약·초기화

| 조작 | 단축키/명령 | 비유 |
|------|------------|------|
| **중단** | `Escape` | "잠깐, 그만!" — Claude가 작업 중일 때 즉시 멈춤 |
| **대화 요약** | `/compact` | "지금까지 한 얘기 정리해줘" — 긴 대화를 요약해서 컨텍스트 확보 |
| **대화 초기화** | `/clear` | "새로 시작하자" — 대화 내용 완전 초기화 |

> 💡 `/compact`는 대화가 길어져서 Claude가 앞부분을 잊어버릴 때 유용합니다. 요약본을 만들어서 기억을 유지합니다.

## EXECUTE

아래 순서대로 실행해보세요:

**실습 1: 캠페인 브리프 템플릿 만들기 (Write)**

Claude Code에 이렇게 요청하세요:

    campaigns 폴더를 만들고, 그 안에 template.md 파일을 만들어줘.
    내용은 기업마케팅 캠페인 기획서 기본 구조로 해줘:
    1. 타겟 기업 과제 분석
    2. 캠페인 KPI
    3. 캠페인 플로우 설계
    4. 콘텐츠 구성
    5. 캠페인 일정·예산
    6. 예상 ROI

**실습 2: 템플릿을 고객 맞춤으로 수정 (Read + Edit)**

    방금 만든 template.md를 읽어서, 대기업 AI 교육 리드 캠페인용으로 수정해줘.
    타겟 기업 과제는 "전사 디지털 전환 가속화를 위한 임직원 AI 역량 강화"로 해줘.

**실습 3: 파일 검색 (Glob)**

    campaigns 폴더에 어떤 파일들이 있는지 보여줘

**실습 4: 내용 검색 (Grep)**

    campaigns 폴더에서 "AI 교육" 키워드가 들어간 파일과 해당 줄을 찾아줘

**실습 5: Plan 모드로 큰 작업 계획하기**

Shift+Tab을 2번 눌러 Plan 모드로 전환한 후 요청하세요:

    campaigns 폴더의 template.md를 기반으로, 3개 고객사(삼성전자, LG전자, SK텔레콤)의 캠페인 기획서를 각각 만들어줘.

Claude가 바로 실행하지 않고 **계획을 먼저 보여줍니다**. 계획을 검토한 후 승인하세요.

**실습 6: 대화 흐름 제어 체험**

아래 3가지를 순서대로 해보세요:

1. Claude가 긴 답변을 하는 중에 **Escape**를 눌러 중단해보세요
2. `/compact`를 입력해서 지금까지의 대화를 요약해보세요
3. `/clear`는 대화를 완전히 초기화합니다 — **지금은 실행하지 마세요!** (학습 진도가 사라집니다). 새 주제로 처음부터 시작할 때 사용하세요.
