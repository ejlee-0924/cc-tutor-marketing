# CC Tutor Marketing — Context

## 프로젝트 개요

B2B 마케터(팀장)를 위한 Claude Code 학습 스킬.
cc-tutor-b2b(기업교육 컨설턴트 버전)에서 마케팅 맥락으로 변환.

## 핵심 설계 결정

1. **cc-tutor-b2b 기반 변환** — 동일한 STOP Protocol, 10모듈 구조 유지
2. **마케팅 맥락 치환** — 제안서→캠페인 기획서, 강사→콘텐츠 소재, 고객→리드
3. **B2B 마케팅 시나리오** — 리드 마그넷, CRM, 웨비나, LP 운영 맥락
4. **진도 JSON 파일** — .cc-tutor-progress.json 동일 스키마
